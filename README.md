Pomelo-Flash-Demo
================

该Demo是基于下面两个框架：

- [Pomelo](https://github.com/NetEase/pomelo)
- [PomeloFlashTCP](https://github.com/D-Deo/pomelo-flash-tcp)

使用步骤：

1. npm install pomelo -g
2. cd server && pomelo start && node crossdomain.js
3. 使用flash里的前端代码运行即可

实现功能：

1. 登陆
2. 显示用户列表
2. 聊天
3. 跟某人聊天
4. 退出
